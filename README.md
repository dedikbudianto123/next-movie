## How To

### Run Development

You can run using npm / yarn

```
yarn run dev
```

### Build Asset

To build new asset using webpack

```
yarn run build
```

### Testing

To run the unit test, you can type this command in terminal.

```
npm run test
```

To run the linter, you can type this command in terminal.

```
npm run build
```
